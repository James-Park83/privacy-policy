계정 및 데이터 삭제 안내

안녕하세요, [금속창호견적]을 이용해 주셔서 감사합니다.
사용자 개인정보 보호를 위해 아래와 같이 계정 및 관련 데이터 삭제 방법을 안내드립니다.

🔐 삭제 가능한 정보

  • 회원 가입 시 등록한 이메일 또는 소셜 로그인 정보
  • 앱 내 활동 기록 (자유게시판 글, 댓글 등)
  • 서비스 이용 중 자동 수집된 기기 정보 (기기 ID, 접속 기록 등)

📝 삭제 요청 방법

1. 앱 내에서 요청하기

앱 실행 > 설정 > 계정 삭제 요청 메뉴 선택
간단한 본인 확인 후 삭제가 진행됩니다.

2. 이메일로 요청하기

아래 이메일 주소로 삭제 요청 메일을 보내주세요.
메일 제목: [금속창호견적] 계정 삭제 요청
